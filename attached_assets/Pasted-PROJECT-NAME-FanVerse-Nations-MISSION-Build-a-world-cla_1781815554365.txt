PROJECT NAME: FanVerse Nations

MISSION

Build a world-class football fan community platform where supporters join their favorite national team, compete for Nation XP, participate in predictions, debates, challenges, and Nation Wars, and help their nation become the strongest fan community in the world.

The app should feel like a combination of FIFA World Cup, Discord, Reddit, Duolingo streaks, Clash of Clans progression, and social gaming.

DESIGN STYLE

* Premium dark mode
* Apple-quality UI
* FIFA World Cup inspired visuals
* Stadium atmosphere
* Glassmorphism cards
* Smooth animations
* Mobile-first design
* Modern typography
* Country flags and nation branding
* Gold, silver, and bronze ranking effects

CORE USER FLOW

1. User signs up.
2. User selects one nation.
3. User joins that nation's fan community.
4. User earns XP through activities.
5. User contributes XP to their nation.
6. User climbs fan rankings.
7. Nation competes against other nations.
